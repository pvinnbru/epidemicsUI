Property Window

Click on the different input sections to obtain information in this window.

Simulation

Define initial simulation settings and the termination criteria.

Pathogens

The user provides a description of the pathogen properties here.

Settings

Here the user can configure parameters indicating the average number of contacts per tick in different settings.

Vaccines

Configure the effectiveness and distribution strategy of vaccinations.

Interventions

Configure how intervention strategies shall be executed in the simulation.


Seed

The seed is an initial value that determines the starting point to generate random data in the simulation. Providing an integer here ensures reproducibility, as the same seed will yield the same simulation results each time.

Unit

A Unit refers to the scale used to quantify time intervals in the simulation. It could be seconds, minutes, hours, days or weeks.

Type (Start Condition)

Specifies the conditions under which the simulation will start. The "Standard" option includes a patient zero, which starts the simulation of the disease outbreak with a single infected individual. With "infected fraction", a certain percentage of the population is infected at the start of the simulation

Pathogen

Specifies the name of the pathogen with which the population has to be infected.

Fraction

If the start criterion is based on a infected fraction, users can input the value for the corresponding percentage here.

Type (Stop-Criterion)

Determines the criteria for stopping the simulation. Users can choose the stopping condition, such as when a specific time is reached.

Limit

If the stop criterion is based on a limit, users can input the value for that limit here.

InfectionRate

The Infection Rate represents the likelihood or probability of an individual getting infected within a specific time frame. It is often associated with the transmission dynamics of the pathogen in the simulated environment. Users define a probability distribution to model how likely an interaction will result in infection.

Time to recovery

Time To Recovery is the duration it takes for an infected individual to recover from the illness. Users define a probability distribution to capture the variability in recovery times among the simulated population. This parameter helps model the recovery phase of the infection.

Mild death rate

The Mild Death Rate signifies the probability of death for individuals who experience mild symptoms during the course of the infection. Users set a probability distribution to simulate the likelihood of death among those with less severe manifestations of the illness.

Critical death rate

The Critical Death Rate represents the probability of death for individuals who develop severe or critical symptoms. Users define a probability distribution to model the fatality risk among those facing more severe consequences of the infection.

Normal Distribution

Symmetrical distribution with a bell-shaped curve. It is characterized by a mean and standard deviation, representing a wide range of continuous random variables.

Uniform Distribution

All values within a specified range are equally likely. It has a constant probability density function.

Exponential Distribution

Models the time between events happening continuously and independently. It's often used when events occur at a constant average rate. The distribution is skewed to the right, and it's commonly used to predict the time until the next event

Poisson Distribution

Models the number of events occurring in a fixed interval of time or space. It is often used for rare events with a known average rate.

Binomial Distribution

Models the number of successes in a fixed number of independent trials, each with the same probability of success.

Gamma Distribution

Continuous probability distribution that generalizes the exponential distribution. It is often used to model the time until a Poisson process reaches a certain number of events.

Beta Distribution

Continuous probability distribution defined on the interval [0, 1]. It is often used to model the distribution of random variables representing proportions or probabilities.

Weibull Distribution

Continuous probability distribution often used to model the distribution of survival times. It can have different shapes, including exponential and normal distributions as special cases.

Age Groups

The user has the option of creating different age groups and assigning the probability of disease progression in the symptom categories "Asymptomatic", "Mild", "Severe" and "Critical" to these age groups.

The stratification matrix is used to assign probabilities to the different symptom categories. By using the sliders, the probabilities are automatically split. The colors represent the different severity of the symptoms.

Household

The Contactparameter indicates the average number of contacts per tick in a Household.

Workplace Contactparameter

The Contactparameter indicates the average number of contacts per tick in a Workplace.

Waning

The user can configure the waning of immunity. Discrete Waning considers the agent as completely immune for a fixed duration and completely susceptible for the rest of the time.

The time to effectiveness describes the fixed time till the vaccine takes effect.

The duration is the time for which the vaccine is effective.

Strategy

Definition of the vaccination strategy in the simulation. Currently only a very simple Daily Dose Strategy is implemented to distribute a fixed number of doses per day.

Under "Avalilable From:" you need to enter the tick unit from which the vaccine should be available.

The Dose Field defines the amount of vaccines that are distributed per time unit.
